
## Example JS Usage
##### Grab a reference  
`const CleverTap = require('clevertap-react-native');`

##### Record an event  
`CleverTap.recordEvent('testEvent');`

##### Update a user profile  
`CleverTap.onUserLogin({'Name': 'testUserA1', 'Identity': '123456', 'Email': 'test@test.com', 'custom1': 123});`
